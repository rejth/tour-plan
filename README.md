**Tour Plan**  
Учебный проект для практики верстки

**Технологии:** 
1) HTML/CSS/JavaScript, Flexbox Layout, CSS Grid Layout, JQuery 
2) Препроцессор SASS 
3) Методология БЭМ 
4) Yandex Map API 
5) Автоматизация и сборка с помощью Gulp

**Функциональность:** 
1) Модальные окна 
2) Анимация
3) Валидация форм 
4) Отправка писем на почту
5) Слайдер
6) Параллакс-эффект
